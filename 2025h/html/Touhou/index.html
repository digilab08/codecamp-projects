Hier ist eine Anleitung zum Touhou Project

Touhou ist eine japanische Videospielreihe von ZUN, bekannt für herausfordernde "Bullet Hell"-Shooter, bei denen Spieler feindlichen Projektilen ausweichen. Die Spiele spielen in der fiktiven Welt Gensokyo, bevölkert von übernatürlichen Wesen. Neben den Spielen ist Touhou berühmt für seine Musik und eine große Fangemeinde, die zahlreiche Fanwerke wie Musik, Kunst und Geschichten schafft.
